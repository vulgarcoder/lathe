# Miniapps

Generator for minxing365's miniapps developer.

## Installation

Add this line to your application's Gemfile:

    gem 'miniapps'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install miniapps

## Usage

miniapps new miniapp_study
